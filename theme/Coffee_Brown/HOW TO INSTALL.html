» HOW TO INSTALL

Tip: Want a hassle-free, always up-to-date help desk with additional features? Upgrade to Hesk Cloud

 

  1. Make sure your Hesk version is compatible with this style.

    Find your Hesk version in Hesk Admin panel > Settings > Checking Status > HESK Version.

     

  2. Backup your existing Hesk files.

    Please do not skip the backup. It will only take you a few minutes and save you hours of headaches if something goes wrong. If anything fails, use the backup to restore your help desk.

     

  3. Upload all included files to your Hesk root directory in CSS folder (overwrite existing files).

     

  4. Make sure your debug mode is turned off in Settings > HelpDesk (for rounded corner)

     

  5. Open your help desk in your browser and force-refresh it to load the new style files.

    In most browsers, open the help desk, then press CTRL + F5 or clear browser cache.

     

Congratulations, your new Admin Hesk Style is installed!
