» Genel Ayarlar

Web Sitesi Başlığı

Web sitenizin kısa başlığı, Örneğin : "Güzel Web Sitem". Bu, web sitenize geri dönüşleri sağlamak için müşteri arayüzünün sol üst köşesinde görüntülenen bağlantının başlığı olarak kullanılacaktır.

Web Sitesi URL Adresi

Web Sitenizin site adresi (URL), Örneğin : "http://www.sitenizinadi.com". Bu adresin geçerli bir URL olduğundan emin olun ve bunu http:// veya https:// ile başlatın. Bu adres, müşteri arayüzünün sol üst köşesinde görüntülenen bağlantının URL'si olarak kullanılacak ve kullanıcıların web sitenize geri dönmelerini sağlayacaktır.

Web Site Yöneticisinin E-Posta Adresi

Varsayılan web yöneticisinin e-posta adresi. Geçerli bir e-posta adresi girdiğinizden emin olun, çünkü herhangi bir site hatası durumunda belirttiğiniz bu adrese mesaj gönderilecektir. Bu adres Destek E-Postası ile aynı olabilir.

 

Pencereyi Kapat

 

"Kimden :" E-Posta Adresi

"Kimden :" e-posta adresiniz. Bu adres, HESK tarafından gönderilen tüm e-postalarda e-postanın göndereni (Kimden : Başlık) olarak gösterilecek e-posta adresidir. Genellikle okumadığınız ve yanıtlamadığınız bir e-postadır. Geçerli bir e-posta adresi girdiğinizden emin olun, bu Destek E-Postası ile aynı olabilir ancak farklı bir e-posta kullanmanız önerilir.

"Kimden :" Adı ve Soyadı

Bu isim, HESK tarafından gönderilen tüm e-postalarda e-postanın göndereni (Kimden : Adı ve Soyadı) olarak gösterilecek addır. Herhangi bir ad ayarlanmazsa, e-posta Gönderen : başlığı yalnızca Kimden : e-posta adresinizi gösterir.

Site Teması (Stili)

Bu, müşteriye yönelik tüm HESK sayfalarında kullanılacak temadır. Düzgün yüklenen tüm temalar açılır liste kutusunda görüntülenecektir. "Tema Klasörünü Test Et" butonu çalışmayan temalardaki sorunu gidermenize yardımcı olur.

Yönetim Paneli Özel CSS Dosyası

Burada, yönetim paneline dahil edilecek özel bir CSS dosyasının URL adresini belirtebilirsiniz.

HESK'in varsayılan yönetim paneli stilinin üzerine yazmak için özel CSS dosyanızı kullanabilir veya kaynak dosyalarını değiştirmeden ek CSS kuralları ekleyebilirsiniz.

Yönetim Paneli Özel JS Dosyası

Burada Yönetici Paneline eklenecek özel bir Javascript dosyasının URL adresini belirtebilirsiniz.

HESK varsayılan Yönetici Paneli Javascript dosyasının üzerine yazmak veya kaynak dosyalarını değiştirmeden ek Javascript eklemek için özel Javascript dosyanızı kullanın.

» DİL AYARLARI

Varsayılan Dil

Hesk için kurulu birden fazla diliniz varsa, bu varsayılan dil olacaktır. Düzgün yüklenen tüm diller açılır liste kutusunda görüntülenir. Daha fazla dilin nasıl kurulacağına ilişkin bilgiler için lütfen readme.html dosyasına bakın. "Dil Klasörünü Test Et" bağlantısı, çalışmayan dil kurulumlarında sorunu gidermenize yardımcı olacaktır.

Birden fazla dil etkinleştirildiyse, tercih ettiğiniz dili "Profil" sayfasında ayarlayabilirsiniz.

Çoklu Dil Kullanımı

Bu ayarın etkinleştirilmesi, kullanıcılarınızın ve personelinizin yüklü diller listesinden tercih ettikleri dili seçmelerine olanak tanır. Tüm kurulu dillerde destek sağlamadığınız sürece bu özelliği etkinleştirmeyin!

» Veri Tabanı Ayarları

Veri Tabanı Sunucusu

Veritabanı ana sunucusu "localhost" çoğu kullanıcı için çalışır. MySQL sunucunuzun ne olduğunu bilmiyorsanız bunu DEĞİŞTİRMEYİN. Emin değilseniz veya MySQL veritabanına bağlanmakta sorun yaşıyorsanız, barındırma şirketinize sorun.

Veri Tabanı Adı

MySQL veritabanınızın adı. Bu bilgiyi web hosting kontrol panelinizde oluşturmalı veya hosting şirketinizden istemelisiniz.

Veri Tabanı Bağlantısı Kullanıcı Adı

MySQL veritabanınıza bağlanmak için kullandığınız kullanıcı adı. Web barındırma kontrol panelinizde bir MySQL kullanıcısı oluşturmalı veya bu bilgiyi barındırma şirketinizden istemelisiniz.

 

Pencereyi Kapat

 

Veri Tabanı Bağlantısı Kullanıcı Parolası

MySQL veritabanına bağlanmak için kullandığınız parola. Web barındırma kontrol panelinizde bir MySQL kullanıcısı oluşturmalı veya bu bilgiyi barındırma şirketinizden istemelisiniz.

Veri Tabanı Tablo Öneki

Tüm tablo adlarına eklenecek önektir.
Örneğin : Bu değer hesk_ olarak ayarlanırsa, "users" tablosu veritabanınızda "hesk_users" olarak adlandırılır.
Bir MySQL veritabanı kullanarak birden fazla Hesk yüklemesi çalıştırmak istiyorsanız, her kuruluma benzersiz bir tablo öneki adı verin (hesk1_, hesk2_, hesk3_ gibi)
