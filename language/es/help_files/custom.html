Ayuda interactiva para la configuración de Hesk

 

» Campos personalizados

Los campos personalizados se pueden utilizar para recopilar información adicional de sus clientes. Puede habilitar hasta 20 campos personalizados seleccionando SÍ en la columna Habilitar. Una vez que un campo está habilitado, tiene varias opciones para configurar.

» Tipo
Elija el tipo de campo. Puede ser una de las siguientes opciones:

  • Cuadro de texto - Un campo de texto normal de una línea (<input type="text">)
  • Cuadro de texto grande - Área de texto, un campo de texto más grande con varias líneas (<textarea></textarea>)
  • Botón de opción - Un botón de radio con al menos dos opciones (<input type="radio">)
  • Cuadro de selección - Un cuadro de selección desplegable con al menos dos opciones (<select></select>)
  • Casilla de verificación - Una casilla de verificación con al menos dos opciones (<input type="checkbox">). Se pueden elegir varias opciones.

» Obligatorio
Marque para hacer que el campo personalizado sea obligatorio, de lo contrario, será opcional.

» Nombre del campo
Asigne al campo personalizado un nombre único que se mostrará junto al campo, por ejemplo "Dirección Postal".

» Ubicación
Elija si el campo personalizado debe mostrarse antes o después de "Campo del mensaje" al enviar un nuevo ticket de soporte.

» Opciones
Opciones que puede configurar para cada campo según el tipo seleccionado:

  • Cuadro de texto - Puede establecer la longitud máxima de entrada (en caracteres) y el valor predeterminado
  • Cuadro de texto grande - Puede establecer el número de filas y columnas (<textarea rows="Y" cols="X">)
  • Botón de opción - Enumera las opciones del botón de opción, una por línea
  • Cuadro de selección - Enumera las opciones del cuadro de selección, una por línea
