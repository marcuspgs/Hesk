Ayuda interactiva para la configuración de Hesk

 

» Parámetros generales

Título del sitio Web

Un título breve de su sitio web, por ejemplo, "Mi encantador sitio web". Esto se usará como el título del enlace que se muestra en la esquina superior izquierda de la interfaz del cliente para permitirles regresar a su sitio web.

URL del sitio Web

URL de su sitio web, por ejemplo "http://www.example.com". Asegúrese de que ésta sea una URL válida y de que comience con http:// o https:// . Esto se usará como la URL del enlace que se muestra en la esquina superior izquierda de la interfaz del cliente y les permitirá regresar a su sitio web.

Correo electrónico del webmaster

Dirección de correo electrónico predeterminada del webmaster. Asegúrese de introducir una dirección de correo electrónico válida, puede ser la misma que la del correo electrónico de soporte.

"De:" email

Su "De:" dirección de correo electrónico. Esta es la dirección de correo electrónico que se mostrará como remitente del correo electrónico (De: encabezado) en todos los correos electrónicos enviados por HESK. Por lo general, es un correo electrónico que no lee y no responde. Asegúrese de introducir una dirección de correo electrónico válida, puede ser la misma que el correo electrónico de soporte, pero se recomienda usar una diferente.

"De:" name

Este es el nombre que se mostrará como remitente del correo electrónico (De:) en todos los correos electrónicos enviados por HESK. Si no se establece ningún nombre, el encabezado De: del correo electrónico solo mostrará su dirección de correo electrónico De:.

Tema del sitio

Este es el tema que se utilizará en todas las páginas HESK orientadas al cliente. Todos los temas correctamente instalados se mostrarán en el cuadro desplegable. La "Carpeta de tema de prueba" le ayudará a solucionar problemas de temas que no funcionan.

Administrar CSS

Aquí puede especificar la URL de un archivo CSS personalizado que se incluirá en el panel de administración.

Utilice su archivo CSS personalizado para sobrescribir el estilo del panel de administración predeterminado de HESK o agregue reglas CSS adicionales sin modificar los archivos de origen.

» Configuración del idioma

Idioma predeterminado

Si tiene varios idiomas instalados para Hesk, este será el idioma predeterminado. Todos los idiomas correctamente instalados se muestran en el cuadro desplegable. Consulte el archivo readme.html para obtener instrucciones sobre cómo instalar más idiomas. La "Carpeta de idioma de prueba" El enlace lo ayudará a solucionar problemas de instalaciones en idiomas que no funcionan.

Si se habilitan varios idiomas, puede configurar su idioma preferido en la página de "Perfil"

Múltiples idiomas

Habilitar esta configuración permitirá a los usuarios y al personal elegir su idioma preferido de la lista de idiomas instalados. ¡No habilite esta característica a menos que brinde asistencia en todos los idiomas instalados!

» Configuración de base de datos

Host de la base de datos

Servidor de alojamiento de base de datos, "localhost" funciona para la mayoría de los usuarios. NO cambie esto a menos que sepa cuál es su servidor MySQL. Pregúntele a su empresa de alojamiento si no está seguro o si tiene problemas para conectarse a la base de datos MySQL.

Nombre de la base de datos

El nombre de su base de datos MySQL. Debe crear una en su panel de control de alojamiento web o solicitar esta información a su empresa de alojamiento.

Usuario de la base de datos

El nombre de usuario que utiliza para conectarse a su base de datos MySQL. Debe crear un usuario de MySQL en el panel de control de su alojamiento web o solicitar esta información a su empresa de alojamiento

 

Cerrar ventana

 

Contraseña de la base de datos

La contraseña que utiliza para conectarse a la base de datos MySQL. Debe crear un usuario de MySQL en el panel de control de su alojamiento web o solicitar esta información a su empresa de alojamiento

Prefijo de tabla

El prefijo que se agregará a todos los nombres de las tablas. Si se establece en la tabla hesk_ "usuarios" en realidad se llamará "hesk_users" en su base de datos. Esto es útil si desea ejecutar varias instalaciones de Hesk usando una base de datos MySQL, solo asigne a cada instalación un nombre de prefijo de tabla único (hesk1_, hesk2_, hesk3_, ...)
