Habilitar base de conocimientos

Utilice esta configuración para activar o desactivar la función HESK Knowledgebase.

Las opciones disponibles son:

  • SÍ, habilite la base de conocimiento
    Habilita la funcionalidad de la base de conocimiento para HESK
     
  • SÍ, use HESK solo como una base de conocimientos (deshabilitar la mesa de ayuda)
    Habilita la base de conocimientos, pero deshabilita la funcionalidad de la ayuda de escritorio (publicar nuevos tickets de soporte). Utilice esta opción si desea utilizar HESK como una base de conocimiento, no como una ayuda de escritorio con todas las funciones.

    Tenga en cuenta que los tickets de soporte aún pueden ser enviados por el personal o por canalización de correo electrónico/obtención de POP3 si está habilitado. Los clientes pueden ver y responder a los tickets.
     
  • NO, deshabilite la base de conocimiento
    Deshabilita la funcionalidad de la base de conocimiento para HESK

Editor WYSIWYG

Esto habilitará un tipo Lo que ves es lo que obtienes editor (un editor de texto enriquecido) para los artículos de la base de conocimiento en lugar de un simple editor de texto.

Sugerir artículos de base de conocimiento

Si se establece en SÍ, Hesk comparará el asunto del ticket y el mensaje con los artículos de la base de conocimientos y recomendará a los clientes que lean los artículos coincidentes antes de enviar un nuevo ticket de soporte.

 

Cerrar ventana

 

Habilitar calificación

Establézcalo en SÍ para permitir que los clientes califiquen los artículos de la base de conocimientos como útiles o no útiles

Habilitar búsqueda

Si habilita esta función se mostrará un formulario de búsqueda en la parte superior de la interfaz del cliente del soporte de escritorio y les permitirá usar el formulario de búsqueda para buscar en su base de conocimientos.

Puede elegir mostrar un cuadro de búsqueda pequeño (ubicado en la esquina superior derecha) o uno grande (más visible, cubre todo el ancho de la página).

 

Cerrar ventana

 

Mostrar vistas del artículo

Establézcalo en SÍ para mostrar públicamente el número de vistas de artículos de la base de conocimiento (las vistas siempre están visibles desde el panel de control del personal).

Mostrar fecha del artículo

Establézcalo en SÍ para mostrar públicamente la fecha de envío del artículo de la base de conocimientos (la fecha siempre está visible desde el panel de control del personal).

 

Cerrar ventana

 

Resultados máximos de búsqueda

Este es el máximo de resultados coincidentes que Hesk devolverá cuando alguien busque en la base de conocimiento.

Longitud de la vista previa del artículo

Longitud máxima (cantidad de caracteres) del contenido del artículo de la base de conocimiento que se mostrará en la lista de artículos al explorar las categorías de la base de conocimiento.

 

Cerrar ventana

 

Categorías en fila

Número de (sub)categorías que se mostrarán en la fila de la tabla al navegar por la base de conocimientos.

Artículos de subcategoría

Número de artículos (vista previa) enumerados en la visualización de subcategoría.

 

Cerrar ventana

 

Mostrar artículos populares

Número de artículos populares (los más visitados) que se muestran en el índice de la mesa de ayuda y índice de la base de conocimientos páginas. Establézcalo en 0 para deshabilitar la lista de artículos populares.

Mostrar artículos más recientes

Número de artículos más recientes (enviados más recientemente) que se muestran en el índice del soporte de escritorio y índice de la base de conocimientos páginas. Establézcalo en 0 para deshabilitar la lista de artículos más recientes.

Artículos relacionados

Establezca la cantidad de artículos relacionados que deben enumerarse debajo de cada artículo de la base de conocimiento.

Para deshabilitar la lista de artículos relacionados, configúrelo en 0
