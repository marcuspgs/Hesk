Aide interactive pour les paramètres Hesk

 

» Champs personnalisés

Les champs personnalisés peuvent être utilisés pour recueillir des informations supplémentaires auprès de vos clients. Vous pouvez activer jusqu'à 20 champs personnalisés en sélectionnant OUI dans la colonne Activer. Une fois qu'un champ est activé, vous avez plusieurs options à définir.

» Type
Choisissez le type de champ. Il peut s'agir de l'une des options suivantes :

  • Champ de texte - un champ de texte normal d'une ligne (<type d'entrée="texte">)
  • Grande zone de texte - zone de texte, un champ de texte plus grand avec plusieurs lignes (<zone de texte></zone de texte>)
  • Bouton radio - un bouton radio avec au moins deux options (<type d'entrée="radio">)
  • Sélectionner la case - une zone de sélection déroulante avec au moins deux options (<sélectionner></sélectionner>)
  • Case à cocher - case à cocher avec au moins deux options (<type d'entrée="case à cocher">). Plusieurs options peuvent être choisies (cochées).

» Obligatoire
Cochez pour rendre le champ personnalisé obligatoire, sinon il est facultatif.

» Nom de domaine
Donnez au champ personnalisé un nom unique qui sera affiché à côté du champ, par exemple "adresse postale".

» Emplacement
Choisissez si le champ personnalisé doit être affiché avant ou après le "Message" champ lors de la soumission d'un nouveau ticket d'assistance.

» Options
Options que vous pouvez définir pour chaque champ en fonction du type sélectionné:

  • Champ de texte - vous pouvez définir la longueur d'entrée maximale (en caractères) et la valeur par défaut
  • Grande zone de texte - you can set number of rows and columns (<lignes de zone de texte="Y" cols="X">)
  • Bouton radio - lister les options de bouton radio, une par ligne
  • Sélectionner la case - liste des options de boîte de sélection, une par ligne
