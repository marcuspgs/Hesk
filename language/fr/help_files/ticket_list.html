Aide interactive pour les paramètres Hesk

 

» Champs dans la liste des tickets

Choisissez les champs que vous souhaitez afficher dans la liste des tickets HESK.

Notez que tous les champs personnalisés doivent d'abord être enregistrés avant d'apparaître dans cette liste.

Si aucun champ n'est sélectionné, le champ ID de suivi sera sélectionné automatiquement.

Au moins un de ces trois champs doit être affiché : ID, ID de suivi, Objet
