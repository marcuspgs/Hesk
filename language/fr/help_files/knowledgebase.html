Aide interactive pour les paramètres Hesk

 

» Paramètres de la base de connaissances

 

Activer la base de connaissances

Utilisez ce paramètre pour activer ou désactiver la fonctionnalité HESK Knowledgebase.

Les options disponibles sont :

  • OUI, activer la base de connaissances
    Active la fonctionnalité de la base de connaissances pour HESK
  • OUI, utiliser HESK comme base de connaissances uniquement (désactiver le service d'assistance)
    Active la base de connaissances, mais désactive la fonctionnalité du service d'assistance (publiant de nouveaux tickets de support). Utilisation cette option si vous souhaitez utiliser HESK comme base de connaissances, et non comme service d'assistance complet.

    Notez que les tickets d'assistance peuvent toujours être soumis par le personnel ou par courrier électronique/récupération POP3 si activé. Les clients peuvent consulter les tickets et y répondre.
  • NON, désactiver la base de connaissances
    Désactive la fonctionnalité de la base de connaissances pour HESK

Éditeur WYSIWYG

Cela activera un type Ce que vous voyez est ce que vous obtenez éditeur (un éditeur de texte enrichi) pour les articles de la base de connaissances plutôt qu'un simple éditeur de texte.

Suggérer des articles de la base de connaissances

Si la valeur est OUI, Hesk associera l'objet et le message du ticket aux articles de la base de connaissances et recommandera aux clients de lire les articles correspondants avant de soumettre un nouveau ticket d'assistance.

 

Fermer la fenêtre

 

Activer l'évaluation en Ko

Définir sur OUI pour permettre aux clients d'évaluer les articles de la base de connaissances comme Utiles ou Inutiles

Activer la recherche dans la base de connaissances

L'activation de cette fonctionnalité affichera un formulaire de recherche au-dessus de l'interface client du service d'assistance et leur permettra d'utiliser le formulaire de recherche pour effectuer des recherches dans votre base de connaissances.

Vous pouvez choisir d'afficher un petit champ de recherche (situé dans le coin supérieur droit) ou un grand (plus visible, couvre toute la largeur de la page).

 

Fermer la fenêtre

 

Afficher les vues d'articles

Définissez sur OUI pour afficher publiquement le nombre de vues d'articles de la base de connaissances (les vues sont toujours visibles depuis le panneau de configuration du personnel).

Afficher la date de l'article

Définissez sur OUI pour afficher publiquement la date de soumission de l'article de la base de connaissances (la date est toujours visible depuis le panneau de configuration du personnel).

 

Fermer la fenêtre

 

Résultats de recherche maximum

Il s'agit du nombre maximal de résultats correspondants que Hesk renverra lorsque quelqu'un effectuera une recherche dans la base de connaissances.

Longueur de l'aperçu de l'article

Longueur maximale (nombre de caractères) du contenu de l'article de la base de connaissances qui sera affiché dans la liste des articles lors de la navigation dans les catégories de la base de connaissances.

 

Fermer la fenêtre

 

Catégories en ligne

Nombre de (sous-)catégories à afficher dans la ligne du tableau lors de la navigation dans la base de connaissances.

Articles de sous-catégorie

Nombre d'articles (aperçus) répertoriés dans l'affichage des sous-catégories.

 

Fermer la fenêtre

 

Afficher les articles populaires

Nombre d'articles populaires (les plus visités) affichés sur l'index du service d'assistance et pages d'index de la base de connaissances. Définissez sur 0 pour désactiver la liste des articles populaires.

Afficher les derniers articles

Nombre d'articles les plus récents (les plus récemment soumis) affichés sur l'index du service d'assistance et pages d'index de la base de connaissances. Définissez sur 0 pour désactiver la liste des derniers articles.

 

Fermer la fenêtre

 

 

 

 

Articles connexes

Définissez le nombre d'articles connexes qui doivent être répertoriés sous chaque article de la base de connaissances.

Pour désactiver la liste des articles liés, définissez sur 0
