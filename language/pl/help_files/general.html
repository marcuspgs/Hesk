Interaktywna pomoc dotycząca ustawień Hesk

 

» Ustawienia ogólne

Tytuł witryny

Krótki tytuł Twojej witryny, na przykład „Moja śliczna witryna”. Będzie on używany jako tytuł linku wyświetlanego w lewym górnym rogu interfejsu klienta, aby umożliwić mu powrót do Twojej witryny.

Adres URL witryny

Adres URL Twojej witryny, na przykład „http://www.example.com”. Upewnij się, że jest to poprawny adres URL i zaczyna się od http:// lub https://. Będzie on używany jako adres URL linku wyświetlanego w lewym górnym rogu interfejsu klienta i umożliwiającego powrót do witryny.

 

Zamknij okno

 

Adres e-mail webmastera

Domyślny adres e-mail webmastera. Upewnij się, że podałeś prawidłowy adres e-mail. Może być taki sam jak adres e-mail pomocy technicznej.

 

Zamknij okno

 

Adres e-mail „Od:”

Twój adres e-mail „Od:”. To adres e-mail, który będzie wyświetlany jako nadawca wiadomości (w nagłówku „Od:”) we wszystkich wiadomościach e-mail wysyłanych przez HESK. Zazwyczaj jest to wiadomość, której nie czytasz i na którą nie odpowiadasz. Upewnij się, że podałeś prawidłowy adres e-mail. Może być taki sam jak adres e-mail pomocy technicznej, ale zaleca się użycie innego.

„Od:” name

To jest nazwa, która będzie wyświetlana jako nadawca wiadomości e-mail (Od:) we wszystkich wiadomościach e-mail wysyłanych przez HESK. Jeśli nazwa nie zostanie ustawiona, nagłówek wiadomości e-mail w polu „Od:” będzie zawierał tylko Twój adres e-mail w polu „Od:”.

Motyw witryny

To jest motyw, który będzie używany na wszystkich stronach HESK przeznaczonych dla klientów. Wszystkie poprawnie zainstalowane motywy będą wyświetlane w polu rozwijanym. „Folder testowy motywu” Przycisk pomoże Ci rozwiązać problemy z niedziałającymi motywami.

Archiwum CSS

Tutaj możesz określić adres URL niestandardowego pliku CSS, który zostanie dołączony do panelu administracyjnego.

Użyj swojego niestandardowego pliku CSS, aby nadpisać domyślny styl panelu administracyjnego HESK lub dodać dodatkowe reguły CSS bez modyfikowania plików źródłowych.

Archiwum JS

Tutaj możesz określić adres URL niestandardowego pliku JavaScript, który zostanie dołączony do panelu administracyjnego.

Użyj swojego niestandardowego pliku JavaScript, aby nadpisać domyślny styl panelu administracyjnego HESK lub dodać dodatkowe reguły JavaScript bez modyfikowania plików źródłowych.

» Ustawienia języka

Język domyślny

Jeśli masz zainstalowanych kilka języków dla Hesk, ten będzie językiem domyślnym. Wszystkie poprawnie zainstalowane języki są wyświetlane w polu rozwijanym. Instrukcje dotyczące instalowania kolejnych języków znajdują się w pliku readme.html. Link „Testuj folder językowy” pomoże Ci rozwiązać problemy z niedziałającymi instalacjami językowymi.

Jeśli włączono obsługę wielu języków, możesz ustawić preferowany język w „Profilu”. strona.

Dostępne języki

Tutaj możesz zarządzać dostępnymi pakietami językowymi dla Hesk (wyświetlać, instalować, aktualizować i usuwać pakiety językowe).

Wiele języków

Włączenie tego ustawienia umożliwi użytkownikom i personelowi wybór preferowanego języka z listy zainstalowanych języków. Nie włączaj tej funkcji, chyba że zapewniasz wsparcie we wszystkich zainstalowanych językach!

Host bazy danych
