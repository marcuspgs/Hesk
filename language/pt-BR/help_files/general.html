Ajuda interativa para configurações do Hesk

 

» Configurações gerais

Título do site

Um título curto para o seu site, por exemplo "Meu site adorável". Isso será usado como o título do link exibido no canto superior esquerdo da interface do cliente para permitir que ele retorne ao seu site.

URL do site

URL do seu site, por exemplo "http://www.exemplo.com". Certifique-se de que é uma URL válida e que começa com http:// ou https://. Isso será usado como a URL do link exibido no canto superior esquerdo da interface do cliente para permitir que ele retorne ao seu site.

Email do webmaster

Endereço de email padrão do webmaster. Certifique-se de inserir um endereço de email válido; pode ser o mesmo que o email de suporte.

 

Fechar janela

 

Email "De:"

Seu endereço de email "De:". Este é o endereço de email que será mostrado como remetente do email (cabeçalho From:) em todos os emails enviados pelo HESK. Geralmente é um email que você não lê e não responde. Certifique-se de inserir um endereço de email válido; pode ser o mesmo que o email de suporte, mas é recomendável usar um diferente.

Nome "De:"

Este é o nome que será mostrado como remetente do email (From:) em todos os emails enviados pelo HESK. Se nenhum nome for definido, o cabeçalho From: do email mostrará apenas seu endereço de email "De:".

Tema do site

Este é o tema que será usado em todas as páginas do HESK voltadas para o cliente. Todos os temas instalados corretamente serão exibidos na caixa suspensa. O botão "Testar pasta do tema" ajudará você a solucionar problemas de temas que não funcionam.

CSS do admin

Aqui você pode especificar a URL de um arquivo CSS personalizado que será incluído no painel de administração.

Use seu arquivo CSS personalizado para substituir o estilo padrão do painel de administração do HESK ou adicionar regras CSS adicionais sem modificar os arquivos fonte.

JS do admin

Aqui você pode especificar a URL de um arquivo Javascript personalizado que será incluído no painel de administração.

Use seu arquivo Javascript personalizado para substituir o Javascript padrão do painel de administração do HESK ou adicionar Javascript adicional sem modificar os arquivos fonte.

» Configurações de idioma

Idioma padrão

Se você tiver vários idiomas instalados para o Hesk, este será o idioma padrão. Todos os idiomas instalados corretamente são exibidos na caixa suspensa. Consulte o arquivo readme.html para obter instruções sobre como instalar mais idiomas. O link "Testar pasta de idioma" ajudará você a solucionar problemas de instalações de idioma que não funcionam.

Se vários idiomas estiverem habilitados, você pode definir seu idioma preferido na página "Perfil".

Idiomas disponíveis

Aqui você pode gerenciar os pacotes de idioma disponíveis para o Hesk (listar, instalar, atualizar, remover pacotes de idioma).

Múltiplos idiomas

Ativar esta configuração permitirá que seus usuários e equipe escolham seu idioma preferido na lista de idiomas instalados. Não ative este recurso a menos que você ofereça suporte em todos os idiomas instalados!

» Configurações de banco de dados

Host do banco de dados

Servidor host do banco de dados, "localhost" funciona para a maioria dos usuários. NÃO altere isso, a menos que você saiba qual é seu host MySQL. Pergunte à sua empresa de hospedagem se não tiver certeza ou se estiver tendo problemas para se conectar ao banco de dados MySQL.

Nome do banco de dados

O nome do seu banco de dados MySQL. Você deve criar um em seu painel de controle de hospedagem na web ou perguntar à sua empresa de hospedagem por essas informações.

Usuário do banco de dados

O nome de usuário que você usa para se conectar ao seu banco de dados MySQL. Você deve criar um usuário MySQL em seu painel de controle de hospedagem na web ou perguntar à sua empresa de hospedagem por essas informações.

 

Fechar janela

 

Senha do banco de dados

A senha que você usa para se conectar ao banco de dados MySQL. Você deve criar um usuário MySQL em seu painel de controle de hospedagem na web ou perguntar à sua empresa de hospedagem por essas informações.

Prefixo da tabela

O prefixo que será adicionado a todos os nomes de tabelas. Se definido como hesk_ a tabela "users" será realmente nomeada "hesk_users" em seu banco de dados. Isso é útil se você quiser executar várias instalações do Hesk usando um banco de dados MySQL, basta dar a cada instalação um nome de prefixo de tabela único (hesk1_, hesk2_, hesk3_, ...)
