Ajuda interativa para configurações do Hesk

 

» Campos na lista de tickets

Escolha quais campos você deseja que sejam exibidos na lista de tickets do HESK.

Observe que quaisquer campos personalizados precisam ser salvos primeiro antes de aparecerem nesta lista.

Se nenhum campo for selecionado, o campo ID de Acompanhamento será selecionado automaticamente.

Pelo menos um destes três campos deve ser mostrado: ID, ID de Acompanhamento, Assunto
