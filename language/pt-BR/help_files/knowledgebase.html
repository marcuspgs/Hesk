Ativar Base de Conhecimento

Use esta configuração para ativar ou desativar a funcionalidade de Base de Conhecimento do HESK.

As opções disponíveis são:

  • SIM, ativar Base de Conhecimento
    Ativa a funcionalidade de Base de Conhecimento para o HESK
     
  • SIM, usar HESK apenas como Base de Conhecimento (desativar central de ajuda)
    Ativa a Base de Conhecimento, mas desativa a funcionalidade de central de ajuda (envio de novos tickets de suporte). Use esta opção se desejar usar o HESK como uma Base de Conhecimento, não uma central de ajuda completa.

    Observe que os tickets de suporte ainda podem ser enviados pela equipe ou por email piping/busca POP3 se habilitados. Os clientes podem visualizar e responder aos tickets.
     
  • NÃO, desativar Base de Conhecimento
    Desativa a funcionalidade de Base de Conhecimento para o HESK

Editor WYSIWYG

Isso habilitará um editor do tipo What You See Is What You Get (um editor de texto rico) para os artigos da base de conhecimento em vez de um simples editor de texto.

Sugerir artigos da base de conhecimento

Se definido como SIM, o Hesk corresponderá o assunto e a mensagem do ticket aos artigos da base de conhecimento e recomendará aos clientes que leiam os artigos correspondentes antes de enviar um novo ticket de suporte.

Ativar avaliação da base de conhecimento

Defina como SIM para permitir que os clientes classifiquem os artigos da base de conhecimento como Útil ou Não útil

Ativar pesquisa na base de conhecimento

Ativar este recurso mostrará um formulário de pesquisa no topo da interface do cliente da central de ajuda e permitirá que eles usem o formulário de pesquisa para pesquisar em sua base de conhecimento.

Você pode escolher exibir uma pequena caixa de pesquisa (localizada no canto superior direito) ou uma grande (mais visível, cobre toda a largura da página).

Mostrar visualizações de artigos

Defina como SIM para exibir publicamente o número de visualizações de artigos da base de conhecimento (as visualizações são sempre visíveis no painel de controle da equipe).

Mostrar data do artigo

Defina como SIM para exibir publicamente a data de envio do artigo da base de conhecimento (a data é sempre visível no painel de controle da equipe).

Máximo de resultados de pesquisa

Este é o máximo de resultados correspondentes que o Hesk retornará quando alguém pesquisar na base de conhecimento.

Comprimento da prévia do artigo

Comprimento máximo (número de caracteres) do conteúdo do artigo da base de conhecimento que será exibido na lista de artigos ao navegar pelas categorias da base de conhecimento.

Mostrar artigos populares

Número de artigos populares (mais visitados) mostrados na página inicial da central de ajuda e nas páginas de índice da base de conhecimento. Defina como 0 para desativar a listagem de artigos populares.

Mostrar artigos recentes

Número de artigos recentes (enviados mais recentemente) mostrados na página inicial da central de ajuda e nas páginas de índice da base de conhecimento. Defina como 0 para desativar a listagem de artigos recentes.
